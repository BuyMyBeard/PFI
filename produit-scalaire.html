Produit scalaire

Le produit scalaire est une opération qui se produit entre 2 vecteurs. Elle est représentée par l'équation suivante:

u • v = ||u||·||v||·cos(θ)

Où u et v sont des vecteurs, et θ est l'angle entre ces deux vecteurs. Le produit scalaire (•) ne doit pas être confondu avec une multiplication. Cette opération donne un scalaire, qui sera > 0 si l'angle entre les 2 vecteurs est aigu, < 0 si l'angle est optus, et = 0 si l'angle est orthogonal.

Cette opération est extrêmement utile pour différentes applications, soit de déterminer si un objet est dans le champ de vision d'un personnage. Une autre application complexe du produit scalaire est le bruit de perlin, qui permet de créer une génération aléatoire naturelle, utilisée notamment pour la génération de monde dans Minecraft.
